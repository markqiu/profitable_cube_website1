# 能力概述[](javascript:;)

1. 排除一些可能存在风险的股票，包括特别处理的股票（ST股）和停牌的股票。这是因为这些股票可能存在一些风险，如公司经营状况不佳，或者因为某些原因暂停交易。
2. 筛选一天内跌幅最大的10支股票。这可能是因为市场对这些股票的评价过于悲观，或者是因为一些短期的负面消息导致股价大幅下跌，因此可能存在反弹的机会。
3. 对这10支股票按照股价从低到高进行排序，优先选择股价较低的股票。这是因为股价较低的股票相对来说投资的门槛更低，而且如果这些股票的基本面没有问题，那么它们的股价有可能会反弹。最终，我们选取股价最低的5支股票作为投资目标。

# 实现逻辑[](javascript:;)

本策略是一个基于市场价格波动的投资策略，旨在识别并投资于那些可能被市场暂时低估的股票。关注那些在短期内经历了显著价格下跌的股票，假设：这些股票的价格下跌可能是市场情绪的反应，而非其基本面的真实反映。因此，这些股票可能会在未来的某个时间点反弹。

# 触发方式[](javascript:;)

交易日收盘后

# 算法说明[](javascript:;)

1. 剔除所有ST股和停牌股。
2. 选取日跌幅最大的10支股票。
3. 按照股价从低到高进行优先级排序，优先选择股价较低的股票。最终选取股价最低的5支股票作。
